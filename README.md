#    Проверка гипотезы о длительности использования приложения Яндекс.Книги в г.Москва и г.Санкт-Петербург. 

## Описание проекта

Аналитический проект посвящённый проверке статистических гипотез в бизнес-контексте для `Яндекс. Книги`, в целях сравнения длительности использования приложения в г. Москва и г.Санкт-Петербург. Продуктовая команда предполагает, что в г.СанктПетербург в среднем пользователи больше времени проводят за прослушиванием и чтением книг в приложении по сравнению с жителями города Москва. 

Во время анализа в нашем распоряжении были данные о действиях пользователей и их распределение по группам. В рамках исследования нами были проанализированы результаты тестирования и определена статистическая значимость различий между данными пользователей. 

##  Цель проекта

заключатся в проведении проверки гипотезы о том, что пользователи из Санкт-Петербурга в среднем проводят больше времени за чтением и прослушивание книг в приложении по сравнению с жителями города Москвы. Для проверки будут использоваться различные инструменты, в частности статистически тест, который позволит определить существует ли статистическая значимость различий между средним временем нахождения пользователей в приложении из двух городов.

##  Задачи исследования:
- загрузка и знакомство с данными из `yandex_knigi_data`
- проведение предобработки данных по необходимости ( изменение типов данных, обработка дубликатов и пропусков);
- проверка гипотезы о наличии статистически значимых различиях количества часов активности в приложении Яндекс. Книги пользователей;
- составление аналитической записки и приведение выводов о проделанной работе.

## Данные
данные представлены в  датасете `yandex_knigi_data`, в котором отображены данные об активности пользователей приложения Яндекс. Книги из 2х городов.
Исходные данные не хранятся в репозитории и используются только в агрегированном виде.


 ## Стек технологий и инструментов:
- Python (pandas, numpy, scipy, matplotlib, seaborn)
- Математическая статистика
- Проверка гипотез

## Методология
Применён непараметрический тест Манна-Уитни ввиду отсутствия нормального распределения данных. Уровень значимости принят стандартный — 0,05.

## Что было сделано в проекте:
- проведён анализ исторических данных о времени активности пользователей в приложении;
- осуществлена предобработка данных, удаление дублей и проверка на наличие нормального распределения;
- реализация статистического теста для проверки гипотезы;
- интерпретация результатов тестирования и выводы.

## Результаты и выводы:
- образованы две независимые выборки разделённые  на данные о жителях г.Москва (5990 человек) и г. Санкт-Петербурга ( 2306 человек). Выборки репрезентативны и независимы;
- данные не имеют нормального распределения. что было подтверждено методом Колмогорова-Смирнова  и графическим методом;
- тест Манна-Уитни показал p-value = 0,453, что выше заданного уровня значимости (0,05);
- по результату проведенного теста мы не можем отвергнуть нулевую гипотезу: различия в средней активности пользователей между Москвой и Санкт-Петербургом статистически не значимы;
- практические рекомендации: для дальнейшего исследования стоит сосредоточиться на других характеристиках пользователей и сравнить их поведение в разных регионах. Это поможет выявить реальные причины различий и выработать конкретные рекомендации по развитию приложения.

 **Наше исследование показало, что в настоящее время значимых различий в активности пользователей Яндекс. Книг между Москвой и Санкт-Петербургом не выявлено. **
